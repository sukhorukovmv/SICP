### Упражнение 1.10
Следующая процедура вычисляет математическую функцию, называемую функцией Аккермана. 

> [Code](../../src/chapter1/1.10.rkt)

Каковы значения следующих выражений? 

```
    (A 1 10)
    (A 2 4)
    (A 3 3)
```

#### Решение

```
    (A 1 10) 1024
    (A 2 4) 65536
    (A 3 3) 65536
```
